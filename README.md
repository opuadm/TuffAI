How to build:

Run `make` (yes, that's all).

After it has built, just run the "tuffai" binary or use `make run` instead of `make`.

To choose which models would be compiled in, just do `MODELS=` (like v1 or v2) after `make`. Multiple models are accepted with separating via commas (example: `MODELS=v1,v2`). Not specifying any `MODELS=` argument to make will just compile all models in.
